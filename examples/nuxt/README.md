Fedify-Nuxt integration example
===============================

This is a simple example of how to integrate Fedify into a [Nuxt] app.

Run it with:

~~~~ bash
pnpm install
pnpm dev
~~~~

Then open <http://localhost:3000/> and try looking up
<http://localhost:3000/users/demo> with Fedify CLI.

[Nuxt]: https://nuxt.com/
